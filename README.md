# 📧 AI Email Spam Detector

A Machine Learning-based web application that detects whether an email/message is Spam or Safe (Ham).

## 🚀 Features

* Spam Detection using Machine Learning
* Confidence Score Prediction
* Login Authentication
* Download Prediction History (CSV)
* Spam vs Safe Analytics Chart
* Professional Gradio UI
* Hugging Face Deployment

## 🛠 Technologies Used

* Python
* Scikit-learn
* Gradio
* Pandas
* Matplotlib

## 🌐 Live Demo

Deployed using Hugging Face Spaces

## 📊 Example Output

* 🚨 SPAM EMAIL
* ✅ SAFE EMAIL
